## Getting Started

### Prerequisites

This project requires the following dependencies:

- **Programming Language:** Python

### Installation

Build flow-builder-backend from the source and intsall dependencies:

1. **Clone the repository:**

    ```sh
    ❯ git clone https://github.com/chitrank2050/flow-builder-backend
    ```

2. **Navigate to the project directory:**

    ```sh
    ❯ cd flow-builder-backend
    ```

3. **Install the dependencies:**

 ```sh
    ❯ make init
    ❯ make install
    ```

### Usage

Run the project with:

 ```sh
    ❯ make dev
    ```
